Introduction

  • Hyper Text Markup Language (HTML) is the standard markup language for creating websites.
  • HTML translates what programmer wants from the website by marking elements to describe the structure of it.

Basic HTML Structure

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>
