Introduction

  • Hyper Text Markup Language (HTML) is the standard markup language for creating websites.
  • HTML translates what programmer wants from the website by marking elements to describe the structure of it.

Basic HTML Structure

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

HTML Structure Explained

  • HTML documents start with a document type declaration: <!DOCTYPE html>.
  • The document begins with <html> and ends with </html>.
  • All the information about document's title, styles, scripts, etc. are defined in between <head> and </head> tags.
  • Front-end (visible part) of the document begins with <body> and ends with </body>.
